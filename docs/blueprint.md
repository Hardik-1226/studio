# **App Name**: MediShop

## Core Features:

- Homepage: Display a hero section, featured products, and basic categories.
- Products Page: Show a grid-based product listing with images, names, prices, and 'Add to Cart' buttons.
- Product Details Page: Display detailed product information including image, name, description, and price.
- Cart Management: Manage the shopping cart state using React Context or Zustand and persist data with localStorage.
- Checkout Process: Implement a checkout page with a simple form and Razorpay payment gateway integration (frontend-only).
- Order Confirmation: Show an order success page after payment with a confirmation message.
- Product Recommendations: AI-powered feature which will use current cart content to provide product recommendations. A tool can check localStorage or current session, look for similarities in medicine category or medical use, and suggest new products in cart page

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke a sense of cleanliness and trust.
- Background color: Off-white (#F5F5F5) for a clean and professional look.
- Accent color: Green (#90EE90) for buttons and highlights, symbolizing health.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable feel.
- Use simple and clear icons for navigation and product categories.
- Mobile-first, fully responsive design, ensuring clarity and readability across all screen sizes.
- Avoid animations and heavy effects to maintain a simple and professional look.
